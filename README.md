# Portfólio de Redações – Nicolas Steinck

Projeto com 15 redações (padrão ENEM) prontas para uso em apresentação escolar.
Tema visual: paleta pastel quente (bege, pêssego, creme) e fonte Poppins.

## Estrutura de pastas
- index.html
- css/styles.css
- js/main.js
- assets/images/ (substitua as imagens production1.jpg ... production15.jpg)

## Como usar
1. Extraia o ZIP em sua máquina.
2. Abra a pasta no VSCode.
3. Abra o arquivo `index.html` com Live Server (ou abra no navegador).
4. Edite `assets/images/` para substituir as imagens placeholder.

## Observações
- O formulário de contato simula envio via alert (sem backend).
- Para publicar no GitHub Pages, faça commit dos arquivos em um repositório e ative o Pages nas configurações.
